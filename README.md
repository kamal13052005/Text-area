# Character Counter Feedback Form

A sleek, responsive feedback interface featuring a real-time character counter with dynamic color feedback. Built using HTML, CSS, and Vanilla JavaScript.

Features

* Real-time Tracking:
   Automatically updates the character count as the user types.
  Visual Feedback:
    The counter changes color based on the length of the input:
  Gray:
     Under 150 characters.
  Orange:
    Over 150 characters.
    * **Red**: Reaches the 200-character limit.
* **Responsive Design**: A centered layout using Flexbox that adjusts to the viewport.
* **Input Limitation**: A hard limit of 200 characters enforced via HTML attributes.
